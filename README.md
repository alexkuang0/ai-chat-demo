# AI Chat Demo

AI chat demo with OpenAI JavaScript SDK.

## Usage

```sh
# install dependencies with pnpm
pnpm install

# copy .env file and fill it with OpenAI API keys: https://platform.openai.com/account/api-keys
cp .env.example .env

# run the chat demo
pnpm run dev
```
